10.Write a shell script to print students grades
   Above 90-A
	 80-B
	 70-C
	 60-D
    else fail.
Ans. 
	echo "enter the marks"
	read marks
	if test $marks -gt 90
	then echo "A"
	elif test $marks -gt 80
	then echo "B"	
	elif test $marks -gt 70
	then echo "C"	
	elif test $marks -gt 60
	then echo "D"
	elif echo "FAIL"
fi
